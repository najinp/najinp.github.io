box의 기본 성질(display:block)

  • 투명하다.
  • width: auto / height: auto
  • Y축으로 쌓여간다.
  • block 옆에 block이 오지 못하게 한다.
  1. 태그는 크게 inline / block으로 나뉜다.
  2. 글자 그림 등 내용에 관여하는 태그 span strong em icon input, button, img는 인라인 속성을 가지고 나머지는 block 속성을 가진다.
  3. block 속성은 width, height, margin, padding 값을 가지고, inline 속성은 좌우 margin padding 만을 가진다.
  4. inline 상하 padding의 경우 background를 포함하여 padding이 적용된 것 처럼 보인다.

css는 레이아웃을 만들어준다.
인라인 스타일(Inline style)이 외부/내부 스타일 보다 우선한다. (계급이 높다.)
javascript의 적용범위는 인라인 스타일(Inline style)이므로 외부/내부 스타일 보다 우선한다.
